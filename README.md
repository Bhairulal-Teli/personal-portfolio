# Personal Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and experience as a Full Stack Developer.

[Portfolio Preview](https://personal-portfolio-z2gv.onrender.com/)

## Features

- **Responsive Design** - Fully optimized for mobile, tablet, and desktop
- **Modern UI/UX** - Clean, brutalist design with vibrant colors
- **Project Showcase** - Interactive project cards with image carousels
- **Tech Stack Display** - Visual representation of technical skills
- **Resume Viewer** - Built-in PDF resume viewer with download option
- **Contact Form** - Functional contact form powered by Web3Forms
- **Smooth Animations** - Engaging transitions and hover effects

## Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- React Router DOM
- Lucide React (Icons)

**Tools & Services:**
- Web3Forms (Contact Form)
- Vite (Build Tool)
- Git & GitHub (Version Control)

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Bhairulal-Teli/portfolio-website.git
```

2. Navigate to project directory
```bash
cd portfolio-website
```

3. Install dependencies
```bash
npm install
```

4. Start development server
```bash
npm run dev
```

5. Open browser and visit `http://localhost:5173`
